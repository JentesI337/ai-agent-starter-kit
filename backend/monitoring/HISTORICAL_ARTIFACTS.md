# Historical Artifacts Notice

This directory contains generated benchmark and runtime telemetry snapshots.

These artifacts are historical records and may include legacy wording from earlier runs
(for example, old delegation status strings or legacy profile hints).

Scope clarification:
- Files under `monitoring/benchmarks/**` and `monitoring/benchmarks_live/**` are immutable run outputs.
- Legacy wording in those snapshots does not represent current runtime behavior.
- Product-facing wording and routing semantics are defined by active source files under `backend/app/**` and tested via `backend/tests/**`.
